# Core Module

Shared services, models, guards, and interceptors used across the application.

## Structure

- `services/` - Shared services (API, storage, etc.)
- `guards/` - Route guards (auth guard, permission guard)
- `interceptors/` - HTTP interceptors (auth token, error handling)
- `models/` - TypeScript interfaces and types

## Purpose

Provides core functionality and shared resources for all feature modules.
